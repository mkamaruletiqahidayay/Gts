# 📱 GTS Kewangan Projek — Cara Install sebagai App

## Fail yang diperlukan
- `index.html` — App utama
- `manifest.json` — Maklumat PWA
- `sw.js` — Service Worker (offline support)
- `icon-192.png` — Ikon app (kecil)
- `icon-512.png` — Ikon app (besar)

---

## 🌐 Langkah 1: Upload ke Netlify (PERCUMA)

1. Pergi ke **https://app.netlify.com/drop**
2. Zip semua 5 fail di atas jadi satu fail `gts-app.zip`
3. Drag & drop fail zip tu ke laman Netlify
4. Netlify bagi URL seperti: `https://amazing-name-123.netlify.app`
5. Simpan URL tu — itu alamat app awak!

---

## 📱 Langkah 2: Install di Telefon

### iPhone / iPad (Safari):
1. Bukak URL Netlify di **Safari**
2. Tekan butang **Share** (⬆️) bawah skrin
3. Scroll ke bawah → pilih **"Add to Home Screen"**
4. Nama: `GTS Kewangan` → tekan **Add**
5. Ikon app akan muncul di home screen!

### Android (Chrome):
1. Bukak URL Netlify di **Chrome**
2. Tekan menu titik tiga **(⋮)** atas kanan
3. Pilih **"Add to Home screen"** atau **"Install app"**
4. Tekan **Install** / **Add**
5. Ikon app akan muncul di home screen!
